# AI-Based Ticket Automation System

## Problem Statement
Manual ticket creation is time-consuming and error-prone. This project automates ticket generation using NLP.

## Features
- Automatic ticket categorization
- Priority assignment
- Entity extraction using NLP
- Structured JSON output

## Tech Stack
- Python
- NLP (spaCy, NLTK)
- Machine Learning (scikit-learn)
- Streamlit

## How to Run
1. Clone the repository
2. Install dependencies
3. Run app.py using Streamlit
